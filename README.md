# PlanIt
SWE Project for V Semester

# Installation
1. You must have Node.js and MongoDB installed
    1.1 Node.js: https://nodejs.org/en/
    1.2 MongoDB: https://www.mongodb.com/download-center?jmp=nav#community

1. Run the following scripts in the root project folder
    1.1 npm install --save express
    1.2 npm install --save mongoose
    1.3 npm install --save body-parser
    1.4 npm install --save express-session  //Optional
    1.5 npm install -g nodemon              //Optional

2. Run the following scripts in the front-end folder (Read package.json)
    2.1 npm run build

3. Run the project from the root project folder by:
    3.1 nodemon        (or)
    3.2 npm start      (or)
    3.3 npm app.js
